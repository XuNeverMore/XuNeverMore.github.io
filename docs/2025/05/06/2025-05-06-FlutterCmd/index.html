  1. 依赖版本控制
    • ^1.2.3：允许升级到兼容版本（如 1.x.x，但不包括 2.0.0）。

• any：使用最新版本（不推荐，可能导致冲突）。

• 直接指定版本：1.2.3（严格锁定）。

三、常用第三方包推荐

  1. 网络请求
    • http：轻量级 HTTP 客户端。

    • dio：功能强大的网络库（支持拦截器、文件上传等）。

  2. 状态管理
    • provider：官方推荐的轻量级状态管理。

    • riverpod：provider 的增强版。

    • bloc：基于事件驱动的状态管理。

  3. 路由导航
    • go_router：声明式路由管理。

    • auto_route：代码生成路由。

  4. UI 组件
    • flutter_svg：渲染 SVG 图片。

    • cached_network_image：缓存网络图片。

  5. 本地存储
    • shared_preferences：轻量级键值存储。

    • hive：高性能本地数据库。

  6. 工具类
    • intl：国际化支持。

    • equatable：简化对象比较。
